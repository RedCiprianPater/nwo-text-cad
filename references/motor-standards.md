# Motor Standards Reference

## NEMA Motors

### NEMA 17
- **Mounting**: 31mm x 31mm hole pattern
- **Hole diameter**: M3
- **Body**: 42mm x 42mm
- **Typical shaft**: 5mm diameter
- **Common lengths**: 40mm, 48mm

### NEMA 23
- **Mounting**: 47.14mm x 47.14mm hole pattern
- **Hole diameter**: M4
- **Body**: 57mm x 57mm
- **Typical shaft**: 6.35mm (1/4") or 8mm
- **Common lengths**: 51mm, 76mm, 84mm

### NEMA 34
- **Mounting**: 69.6mm x 69.6mm hole pattern
- **Hole diameter**: M6
- **Body**: 86mm x 86mm
- **Typical shaft**: 14mm
- **Common lengths**: 66mm, 96mm, 126mm

## Servo Motors

### Standard Sizes
| Class | Dimensions | Torque |
|-------|-----------|--------|
| Micro | 20x10x20mm | 0.5 kg-cm |
| Mini | 30x15x30mm | 2 kg-cm |
| Standard | 40x20x40mm | 5 kg-cm |
| Large | 60x30x60mm | 15 kg-cm |

## Mounting Guidelines

- **Clearance holes**: +0.4mm for M3, +0.5mm for M4/M5
- **Counterbore depth**: 2-3mm for screw heads
- **Wall thickness**: 3mm minimum around mounts
- **Ribs**: Add for structural support
